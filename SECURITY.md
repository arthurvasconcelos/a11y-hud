# Security Policy

## Reporting a vulnerability

Please use [GitHub private vulnerability reporting](https://github.com/arthurvasconcelos/a11y-hud/security/advisories/new) to report security issues.

Do not open a public issue for security concerns.

## Scope

`a11y-hud` is a developer tool intended for use in development and staging environments only. It should not be included in production builds. If you are using it in production, that is outside the intended scope and not supported.

## Response

Reported vulnerabilities will be acknowledged within 7 days. A fix or mitigation will be provided as soon as reasonably possible. The reporter will be credited in the release notes unless they prefer to remain anonymous.
