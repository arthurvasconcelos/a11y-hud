Get Started
GitHub
🔍

No extension required

Runs axe-core directly inside your running app — in the browser window, on the page. Works in mobile browsers, WebViews, and any context where extensions can't be installed.

⚡

Framework-agnostic

Vanilla core with thin adapters for React, Vue, Angular, Svelte, and Solid. One Custom Element UI, every framework.

🎨

8 built-in themes

default, light, high-contrast, github-dark, github-light, tokyo-night, solarized-dark. Fully customizable via CSS custom properties.

⌨️

Keyboard mode

Tab-order visualization, focusable element list, and automatic detection of tab-stop violations — without leaving the page.

🔕

Ignore-rules persistence

Mute noisy rules or specific selectors. Ignore lists survive page reloads via localStorage and can be exported or imported as JSON.

📎

Bookmarklet

Inject the HUD into any staging site without touching its source. One drag-to-bookmark, no build config, no extension needed.
